MB9 has developed some tools. These are introduced below:

Exchange Web Provisioning System (WPS)

WPS is a web based tool for easy administration of Exchange Servers. WPS support Exchange 2007/2010 and enables easy administration of multi-tenant hosting scenarios. Below are some of the features supported by WPS.
  • Supports multi-tenant hosting scenarios.
  • Support for Department/Organization Unit based control
  • Granular role-based delegated administration
  • Restrict specified roles by Organizational Units (OU)
  • Easy Management of Mail Box, Distribution Groups and Mail Contacts.
  • Advanced management features for Distribution Groups.
  • Reporting and audit logs 
  • Enhanced security.

DomainControl (Role Based AD Administration Delegation)

    DomainControl is a web based administration tool for Role Based AD Administration. It supports the following features.

    • Support for Department/Organization Unit based delegated administration
    • Creation of customizable provisioning and de-provisioning policies
    • Security Group management
    • OU management at appropriate levels.
    • Reporting and audit logs
    • Reporting management for security audits

XConsole for Admin

    XConsole is a helper utility for Exchange Admins. It supports the following features.

    • GUI Based Administration
    • Create Permission Templates
    • Extract/Copy/Apply/Clone Permission Templates from an Entity
    • Generate Permission reports which can be used for Audits and Compliance checks
    • Export Reports to Excel/PDF 

XConsole for Multi-Tenant Setup

    XConsole is a helper utility for Exchange Administrators to create multi-tenant configuration.

    • Create Initial Configuration
    • Setup Address Book Permission
    • Manage Tenants

    This setup will work well with WPS.

    XMonit (Exchange Monitor)

    XMonit monitors your Exchange Server 2007/2010 Setup. It has the following features

    • GUI based setup and configuration of monitors
    • View monitor status in the GUI and on Status Bar when minimized with notifications
    • Monitor your Exchange Server Status and Health.
    • Nagios connector which enables you to connect your existing Nagios setup and monitor from Nagios.
