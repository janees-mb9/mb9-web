WPS: Web Provisioning System

WPS is a resource provisioning system for Microsoft Exchange 2007 and 2010. This product works with multi-tenant hosted MS Exchange solutions and eases management and provisioning for this setup. WPS is applicable for large companies with departments setup as multi-tenant. WPS is also suitable for Exchange Hosting Providers.

Below are some of the features supported by WPS.
  • Supports multi-tenant hosting scenarios.
  • Support for Department/Organization Unit based control
  • Granular role-based delegated administration
  • Restrict specified roles by Organizational Units (OU)
  • Easy Management of Mail Box, Distribution Groups and Mail Contacts.
  • Advanced management features for Distribution Groups.
  • Reporting and audit logs 
  • Enhanced security.
